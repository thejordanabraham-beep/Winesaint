## The MGA System Context

What the research data does clarify is that Barolo's 2010 MGA system created official sub-zones within the denomination that fall into three categories:

- Entire villages (such as La Morra)
- Specified single vineyards (such as Bussia or Cannubi)
- Multiple specified vineyards consolidated into a single MGA (for example, the Via Nuova vineyard was included within the Terlo MGA)

The system also established that the word "vigna" (vineyard) can only appear as a labeling term if the name of the MGA is also stated.
