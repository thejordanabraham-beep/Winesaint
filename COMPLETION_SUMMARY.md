```bash

```

### Production Build
```bash
npm run build
npm start
```

### Generate Additional Guides
```bash
# Check status
npx tsx scripts/status-dashboard.ts

# Generate specific guides
npx tsx scripts/generate-all-guides.ts --only "region-slug"

# Validate all guides
npx tsx scripts/validate-guides.ts --all
```

### Generate Page Templates
```bash
# Generate all missing pages
npx tsx scripts/generate-page-template.ts --all

# Generate specific page
npx tsx scripts/generate-page-template.ts region-slug
```
